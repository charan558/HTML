• What is CSS?
    • CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation of a document written in HTML.

  • What are the different ways to include CSS in an HTML document?
    • CSS can be included in an HTML document using inline styles, internal stylesheets (using the style tag), or external stylesheets (linked using link the tag).

  • What is the box model in CSS?
    • The box model in CSS describes the layout and rendering of elements in a document. It consists of content, padding, border, and margin.
